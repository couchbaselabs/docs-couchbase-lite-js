== Overview

This page provides compatibility information for Couchbase Lite JavaScript 1.0, including supported browsers, backend server versions, framework compatibility, and feature limitations.

For specific deployment constraints and backend compatibility requirements, see xref:known-limitations.adoc[Known Limitations].

IMPORTANT: Users must be on Sync Gateway 3.3.1 or 4.0.1 and above to sync data with the JavaScript SDK. CORS configuration is required -- see xref:gs-prereqs.adoc#prerequisites[Prerequisites].

[#browser-compatibility]
== Browser Compatibility

See xref:supported-browsers.adoc[] for detailed browser version requirements and feature support matrices.

**Desktop Browsers:** Chrome 90+, Firefox 88+, Safari 14+, Edge 90+

**Mobile Browsers:** iOS Safari 14+, Android Chrome 90+, Android Firefox 88+

**Progressive Web Apps (PWA):** Full support for installable web applications with Service Worker integration


IMPORTANT: Couchbase Lite JavaScript does not currently work with Couchbase Lite Edge Server.
Use Sync Gateway or Capella App Services for data synchronization.
