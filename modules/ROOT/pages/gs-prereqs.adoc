[#supported-browsers]
== Supported Browsers

Couchbase Lite JavaScript requires modern browsers with support for IndexedDB, WebSocket, Web Crypto API, and ES2020 JavaScript features. The SDK supports modern, actively maintained browsers to ensure broad adoption while maintaining technical viability. Older, unmaintained or low adoption browsers such as Internet Explorer and Opera are out of scope.

NOTE: Public Beta Support focuses on Chrome and Safari to surface potential cross-browser issues early.

[#required-browser-apis]
== Required Browser APIs

Couchbase Lite JavaScript requires the following browser APIs:

[%autowidth.stretch]
|===
|API |Purpose |Availability

|IndexedDB
|Local data storage
|All supported browsers

|WebSocket
|Replication with Sync Gateway
|All supported browsers

|Web Crypto API
|Encryption at rest
|All supported browsers

|Promises / async-await
|Asynchronous operations
|All supported browsers (ES2017+)

|Storage API (optional)
|Quota management
|Chrome 48+, Firefox 57+, Safari 15.2+
|===
